# Contributing to PiMine
Thanks for your interest in contributing!
## Ways You Can Help

You can contribute by:

* Fixing bugs
* Improving documentation
* Adding new features
* Improving performance or reliability
* Suggesting ideas via issues

---

## Getting Started

1. Fork the repository
2. Create a new branch:

   ```bash
   git checkout -b feature-name
   ```
3. Make your changes
4. Commit clearly:

   ```bash
   git commit -m "Add feature: description"
   ```
5. Open a Pull Request

---

## Code Guidelines

* Keep code simple and readable
* Follow existing project structure
* Add comments when needed
* Avoid unnecessary dependencies

---

## Reporting Issues

When opening an issue, please include:

* What happened
* Steps to reproduce
* Your OS and environment
* Any error messages

---

## Before Submitting a PR

Please make sure:

* The code builds successfully
* You tested your changes
* The change aligns with the project goals

---

Thank you for helping improve PiMine ❤️
